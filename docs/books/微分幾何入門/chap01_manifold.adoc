=== 問題1.10
問::
多様体としての開区間I=(-pi/2, pi/2)とRが微分同相であることを示せ。 +
(I ⊂ R は開区間(a,b)であればなんでもよいことも簡単にわかる)

**回答**

latexmath:[f: I\to \mathbb{R}]の写像に対して、 latexmath:[f(\theta \in I) = \tan\theta]を考えると、__I__から__R__への微分同相写像であることがわかる。
