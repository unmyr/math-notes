== Example: Atlas of the unit sphere S^2^ (angular coordinates)

The unit sphere _S_^2^ can be expressed with angular coordinates (theta, phi) as:
[stem,latexmath]
++++
S^2 = \{(\cos \phi \sin \theta, \sin \phi \sin \theta, \cos \theta) \in \mathbb{R}^3 | 0 \leq \phi \leq \pi, 0 \leq \theta \leq 2\pi\}
++++

To cover the sphere S^2^, we can consider the following two charts:

latexmath:[U_1]では、極角 latexmath:[\theta_1] は z 軸を中心とした回転を表し、方位角 latexmath:[\phi_1] は x-y 平面における回転を表す。開被覆 latexmath:[U_1] は _S_^2^ から半円 latexmath:[\phi_1=0]を除いたものとなる。また、latexmath:[U_2]では、極角 latexmath:[\theta_2] は y 軸を中心とした回転を表し、方位角 latexmath:[\phi_2] は x-z 平面における回転を表し、開被覆 latexmath:[U_2] は  _S_^2^ から 半円 latexmath:[\phi_2=0]を除いたものとなる。このように、それぞれ基底を異なるものとすることで、2つの開被覆で _S_^2^ 全体が被覆される。
In latexmath:[U_1], the polar angle latexmath:[\theta_1] represents the rotation around the z-axis, and the azimuthal angle latexmath:[\phi_1] represents the rotation in the x-y plane. The open cover latexmath:[U_1] is _S_^2^ with the semicircle latexmath:[\phi_1=0] removed. In latexmath:[U_2], the polar angle latexmath:[\theta_2] represents the rotation around the y-axis, and the azimuthal angle latexmath:[\phi_2] represents the rotation in the x-z plane. The open cover latexmath:[U_2] is _S_^2^ with the semicircle latexmath:[\phi_2=0] removed. By choosing different bases for each, the entire _S_^2^ can be covered with these two open covers.

[stem,latexmath]
++++
\begin{aligned}
U_1 &=  S^2 \setminus \left\{ x^2 + z^2 = 1 \mid 0 \leq x \leq 1, -1 \leq z \leq 1 \right\} \\
U_2 &=  S^2 \setminus \left\{ x^2 + y^2 = 1 \mid -1 \leq x \leq 0, -1 \leq y \leq 1 \right\} \\
\end{aligned}
++++

=== 座標関数 (Coordinate functions)

 latexmath:[U_1] 上の座標関数 f_1 は、z軸を中心とした回転を表す極角 latexmath:[\theta_1] と、x-y 平面における回転を表す方位角 latexmath:[\phi_1] を用いて定義する。方位角は開区間 latexmath:[(0, 2\pi)] において定義することで一対一の写像となる。latexmath:[U_2] 上の座標関数 f_2 は、y軸を中心とした回転を表す極角 latexmath:[\theta_2] と、x-z 平面における回転を表す方位角 latexmath:[\phi_2] を用いて定義し、方位角は開区間 latexmath:[(0, 2\pi)] において定義することで一対一の写像となる。

[stem,latexmath]
++++
\begin{aligned}
f_1(x, y, z): U_1 \to \mathbb{R}^2, & \quad f_1 (x, y, z) \mapsto (\arg(x + iy) \in (0, 2\pi), \arccos z) = (\phi_1, \theta_1) \\
f_2(x, y, z): U_2 \to \mathbb{R}^2, & \quad f_2 (x, y, z) \mapsto  \left(\arg((-x) +iz) \in (0, 2\pi), \arccos y\right) = (\phi_2, \theta_2) \\
\end{aligned}
++++

=== 変換関数 (Transition function)

まず、座標関数の逆関数を求める。latexmath:[U_1] 上の座標関数 f_1 の逆関数は、z軸を中心とした回転を表す極角 latexmath:[\theta_1] と、x-y 平面における回転を表す方位角 latexmath:[\phi_1] を用いて、次のように表される。latexmath:[U_2] 上の座標関数 f_2 の逆関数は、y軸を中心とした回転を表す極角 latexmath:[\theta_2] と、x-z 平面における回転を表す方位角 latexmath:[\phi_2] を用いて、次のように表される。

[stem,latexmath]
++++
\begin{aligned}
f_1^{-1}(\phi_1, \theta_1) & \mapsto (\cos \phi_1 \sin \theta_1, \sin \phi_1 \sin \theta_1, \cos \theta_1) \\
f_2^{-1}(\phi_2, \theta_2) & \mapsto (\cos (\phi_2 + \pi) \sin \theta_2, \cos \theta_2, \sin (-\phi_2 + \pi) \sin \theta_2) \\
&= (-\cos \phi_2 \sin \theta_2, \cos \theta_2, \sin \phi_2 \sin \theta_2) \\
\end{aligned}
++++

上記より、 _U_~1~ ∩ _U_~2~ 上の変換関数は次のように表される。 +
From the above, the transition function on _U_~1~ ∩ _U_~2~ can be expressed as follows.

[stem,latexmath]
++++
\begin{aligned}
(\phi_2, \theta_2) = f_2 \circ f_1^{-1}(\phi_1, \theta_1) &= f_2 (f_1^{-1}(\phi_1, \theta_1)) \\
&= f_2 (\cos \phi_1 \sin \theta_1, \sin \phi_1 \sin \theta_1, \cos \theta_1) \\
&= (\arg((-x) +iz) \in (0, 2\pi), \arccos y)) \\
&= \left(\arg((-\cos \phi_1 \sin \theta_1) +i\cos \theta_1) \in (0, 2\pi), \arccos (\sin \phi_1 \sin \theta_1)\right)
\end{aligned}
++++

=== 変換関数が _C_^∞^級であることの確認 (Checking that the transition function is _C_^∞^)
変換関数の各成分は三角関数・arg・arccos の合成である。
これらはそれぞれ _U_~1~ ∩ _U_~2~ の定義域で _C_^∞^ 級であるため、変換関数は _C_^∞^ 級である。
Each component of the transition function is a composition of trigonometric functions, arg, and arccos. Since these are all _C_^∞^ on the domain of _U_~1~ ∩ _U_~2~, the transition function is _C_^∞^.

実際に、確認すると次のようになる。 +
In fact, upon checking, we have the following.

方位角の成分の滑らかさの検証を行う。 +
Verifying the smoothness of the azimuthal angle component.

latexmath:[\arg((-x) +iz) \in (0, 2\pi)]は、latexmath:[(x, z) \ne (0, 0)]で定義される。 方位角の定義域 latexmath:[0 < \theta_1 < \pi]から latexmath:[\sin \theta_1 > 0 \to |\sin \phi_1 \sin \theta_1| < 1]であり、 _U_~1~ ∩ _U_~2~ 共通部分に latexmath:[\phi_1 = \pm \frac{\pi}{2}]は含まれないため、 latexmath:[arg(z)]は _U_~1~ ∩ _U_~2~ 上で定義される。 +
The function latexmath:[\arg((-x) +iz) \in (0, 2\pi)] is defined for latexmath:[(x, z) \ne (0, 0)]. From the domain of the polar angle latexmath:[0 < \theta_1 < \pi], we have latexmath:[\sin \theta_1 > 0 \to |\sin \phi_1 \sin \theta_1| < 1], and since latexmath:[\phi_1 = \pm \frac{\pi}{2}] is not included in the common part of _U_~1~ ∩ _U_~2~, the function latexmath:[arg(z)] is defined on _U_~1~ ∩ _U_~2~.

ここで、 latexmath:[z = x + iy, \quad \theta = \arg z]とすると、latexmath:[\theta(x,y) = \arg (x, y)]の偏微分は次のようになる。 +
Here, let latexmath:[z = x + iy, \quad \theta = \arg z], then the partial derivatives of latexmath:[\theta(x,y) = \arg (x, y)] are as follows.

[stem,latexmath]
++++
\frac{\partial \theta}{\partial x} = \frac{-y}{x^2 + y^2}, \quad \frac{\partial \theta}{\partial y} = \frac{x}{x^2 + y^2}
++++

latexmath:[x = -\cos \phi_1 \sin \theta_1 \ne 0 \to x^2 + y^2 \ne 0]であり、偏微分は _U_~1~ ∩ _U_~2~ 上で定義される。 +
Since latexmath:[x = -\cos \phi_1 \sin \theta_1 \ne 0 \to x^2 + y^2 \ne 0], the partial derivatives are defined on _U_~1~ ∩ _U_~2~.

極角の定義域 latexmath:[0 < \theta_1 < \pi]から latexmath:[\sin \theta_1 > 0 \to |\sin \phi_1 \sin \theta_1| < 1]であるから、latexmath:[\arccos^{\prime}(x)= -\frac{1}{\sqrt{1-x^2}}]の微分不可能な点を含まないことが分かる。 +
In fact, from latexmath:[0 < \theta_1 < \pi], we have latexmath:[\sin \theta_1 > 0 \to |\sin \phi_1 \sin \theta_1| < 1], which means that the transition function does not include the points where latexmath:[\arccos^{\prime}(x)= -\frac{1}{\sqrt{1-x^2}}] is not differentiable.


**NOTE**: arg(z)の要件

arg(z)は複素微分可能ではない。一方で、多様体の文脈では、複素解析的な「正則性」は一切要求していない。
_S_² の遷移写像で必要なのは latexmath:[\arg: \mathbb{R}^2 \setminus \{0\} \to \mathbb{R}]が実意味で C^∞^ 級であることとなる。つまり、次を満たせば十分である。

* 偏微分が存在して連続
* さらに高階偏微分も存在して連続
